### Installation
##### First step
`git clone https://github.com/hyphenn/vimrc yourfolder`

`cp -R yourfolder/* ~`

##### Vundle
`git clone https://github.com/VundleVim/Vundle.vim.git ~/.vim/bundle/Vundle.vim`

#####  Plugin installation with Vundle : 
`:PluginInstall`

#####  Reload .vimrc
`:so %`
